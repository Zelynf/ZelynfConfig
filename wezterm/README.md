# zelynfNvim
Zelynf's configuration for WezTerm, minimalistic and essentials only

> If you are trying to find the script to download the programs and apply the configuration go to [Zelynf/ZelynfConfig](https://github.com/Zelynf/ZelynfConfig)

# Features
- Includes the [Flexoki Theme by Seteph Ango](https://stephango.com/flexoki)    [GitHub](https://github.com/kepano/flexoki)

## Configuration
- [PowerShell 7](https://github.com/PowerShell/PowerShell) is set as the defaut shell
- [JetBrainsMono](https://www.jetbrains.com/lp/mono/) is set as the default font
- [Flexoki](https://github.com/kepano/flexoki) is set as the color scheme (might not work on linux)
- Transparent Background
- Window decorations and tabs are hidden

## KeyBindings
> Default keybinding have been disabled
- Utilizing the configuration from [@quantoganh](https://github.com/wezterm/wezterm/discussions/3779) to create a Split Pane (CTRL + ;)
- Paste from clipboard keybind (CTRL + SHIFT + V)
